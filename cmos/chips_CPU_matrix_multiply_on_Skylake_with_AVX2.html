CPU matrix multiply on Skylake with AVX2
https://gist.github.com/nadavrot/5b35d44e8ba3dd718e595e40184d03f0

• C(m,n) = A(m,k) * B(k,n) = dot-product

• AVX2: 2 exec ports, each capable of 8-wide Fused Multiply-Add ops per clock
• Skylake runs ~4GHz
• So, theoretical thruput = 4GHz * 8 FP#s * 2 ports * 2 ops = 128Gflops

• 1024 x 1024 matrix multiply, using naive algorithm, takes o(n^3) ops, and we need two ops per element (mult, add). ==> we need 1024^3*2 ~ 2.1Gflops

• So, theoretically we could do 128/2.1 ~ 60 1Kx1K matrix multiplys/second
• Naive algo, written in C, runs ~8Gflops/sec (7% utilization). Huh?
